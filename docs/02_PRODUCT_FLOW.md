# 02_PRODUCT_FLOW.md

## 1. The Session Ritual (Lifecycle)

### Phase 1: The Ritual (Start)
1.  **Inception**: User inputs the "Theme" (Top screen).
2.  **Volition**: Registered Personas signal participation interest (Glow/Dim).
3.  **Selection**: User drags chosen Personas to the "Heptagon Ring".
    - *Constraint*: Max 7 seats.
    - *Option*: Force participation of specific Personas.
4.  **Voice of Heaven**: System selects a Moderator (Voice of Heaven) from remaining pool.
5.  **Ignition**:
    - **Animation**: "The Genesis Sequence" (if new) or "Resonance Join".
    - **Transition**: Avatar Mode -> Core Mode (Holo-Node).

### Phase 2: The Council (Discussion)
1.  **Opening**: Voice of Heaven announces the theme.
2.  **Discussion Loop**:
    - **Orbit In**: Speaker Node moves forward.
    - **Statement**: Text projected as "Data Stream". Voice playback.
    - **Synapse Link**: Laser connects Nodes with agreeing views.
    - **User Action**: Double Tap (Like), Swipe (Agree/Eject), Long Press (Mind Dive).
3.  **Deviation Check**: If off-topic, Voice of Heaven intervenes (Deviation Protocol).

### Phase 3: The Ending (Conclusion)
1.  **Convergence**: User or Moderator triggers "Conclusion". Nodes emit light to center.
2.  **Crystallization**:
    - Log compressed into "Knowledge Crystal".
    - **Color**: Red (Heated) or Blue (Calm) based on sentiment.
3.  **Dissolution**:
    - Holo-Nodes revert to Avatars for a split second.
    - Bow and dissolve into particles.
4.  **Result**: Crystal saved to "Memories". Parameters updated (Neuro-Plasticity).

## 2. Digital Forge (Persona Creation Flow)
1.  **Input**:
    - Name, Role, Relation to User.
    - **Knowledge Source**: Text files, PDF, Web links.
2.  **Genesis Sequence**:
    - **Phase 1**: Skeleton/Neural Network construction (visual).
    - **Phase 2**: Glass Body formulation + Data filling.
3.  **Soul Ignition (Phase 3)**:
    - Eye glow activation.
    - Voice synthesis verification.
4.  **Completion**: Added to Persona List.

## 3. Interaction Flow (Gestures)
- **Navigation**:
    - **Swipe Left/Right (Edge)**: Open Logs / Settings.
    - **Rotary (Wheel)**: Turn the table or rewind time (Log scroll).
- **In-Session**:
    - **Double Tap (Node)**: Send "Like" (Empathy).
    - **Long Press (Node)**: "Mind Dive" (View internal thought process).
    - **Swipe Up (Node)**: Agreed / Encourage.
    - **Swipe Down (Node)**: Eject (Force leave).
