# Handoff Q/A Log
- 毎回のやり取りを Q/A ペアで圧縮して追記する。
- 1エントリは10〜15行以内を推奨する。
- Next は必ず1つだけ書く。
- 脱線しそうな改善は PARK.md に退避する（ここでは修正しない）。
