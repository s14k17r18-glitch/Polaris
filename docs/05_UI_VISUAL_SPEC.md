# 05_UI_VISUAL_SPEC.md

## 1. Design System
- **Theme**: "Cyber / Sci-Fi Research Lab" (Dark, Neon, Glass).
- **Color Palette**:
    - **Base**: `#0A0E17` (Deep Space) - Backgrounds.
    - **Accent**: `#00F0FF` (Cyan Neon) - Active elements, Eyes, Tech.
    - **Secondary**: `#7000FF` (Purple) - Accents, Resonance.
- **Material**: "Glassmorphism" (Blur: 10-20px, Translucency: 80%).

## 2. Screen Layouts

### 2.1 Top Screen (The Gateway)
- **Central Element**: "Nexus-AI" Logo (Geometric, Cyberpunk style).
- **Background**: Faint floating data particles in `#0A0E17` void.
- **Action**: Tap to open "Menu" (Glass panel slides from edge).

### 2.2 The Council View (Session)
- **Layout**: "Heptagon Ring" (7 slots arranged in a 3D circle).
- **Perspective**: First-person view from the User's seat.
- **Data Stream**: Real-time conversation text projected on angled glass panes floating near each Node. No speech bubbles.

## 3. Entity Specification

### 3.1 Avatar Mode (Waiting State)
- **Skin**: Porcelain/Doll-like texture (Perfect smooth).
- **Symmetry**: Absolute "Perfect Symmetry" geometry.
- **Eyes**: "Glow Eyes" (Geometric patterns spinning in iris).
- **Accessory**: "Digital Halo" floating behind head.

### 3.2 Core Mode (Discussion State)
- **Shape**: Hexagonal Prism (Holo-Node).
- **Texture**: Holographic wireframe + Solid core.
- **Internal Symbol**: Floating object representing Persona (e.g., Quill, DNA, Cat Silhouette, Crystal).

## 4. Interaction Model (Gestures)
- **Settings/Menu**: Swipe from **Screen Edge** (Left/Right).
- **Log Review**: Rotate **Virtual Wheel** (Rotary Manipulation) or Swipe down.
- **Agreement**: **Double Tap** on a Node (sends #00F0FF pulse).
- **Insight**: **Long Press** on a Node -> Enters "Mind Dive" overlay.
