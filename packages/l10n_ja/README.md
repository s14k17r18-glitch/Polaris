# l10n_ja

日本語文言（ユーザー可視はここだけ）

## 役割

- **完全日本語固定**（00_PARITY_CONTRACT.md）
- **英語直書き禁止**（UI・エラーメッセージ・ユーザー可視ログ）

## 使用例

```dart
import 'package:l10n_ja/l10n_ja.dart';

Text(L10nJa.appTitle)
```

## チェック

A6 で「英語直書き禁止」の自動チェックを導入予定

## 文言一覧

### アプリ基本

- `appTitle`: 北極星（Hokyoksei）
- `helloWorld`: M0: 骨組み確認用 Hello World
